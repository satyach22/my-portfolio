Project 2: Smart Aquaculture: Predictive Analytics for Water Quality Control

Description: Water pollution detection in aquaculture ponds using machine learning and IoT involves deploying sensors to monitor water quality parameters, such as pH, temperature, and dissolved oxygen. The collected data is analyzed using machine learning algorithms to identify patterns and detect contaminants. This real-time monitoring system helps in maintaining optimal water conditions, ensuring the health and productivity of aquatic life.

Technologies Used: Multiparameter Meter, Data Loggers, Arduino, Python- Scikit Learn.

Project 3: To-Do List App

Description: A To-Do List App is a simple yet effective tool designed to help users organize and manage their tasks. The app allows users to create, view, update, and delete tasks they need to complete. Each task can have additional details, such as deadlines, priorities, and categories. The primary goal is to help users stay organized and productive by giving them a clear view of their to-do list and enabling task tracking.

Technologies Used: Django, PostgreSQL, Bootstrap
